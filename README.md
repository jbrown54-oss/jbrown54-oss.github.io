# Jonas Brown's Personal Website

A professional personal website built with Next.js, featuring a dynamic neural network background that responds to cursor movement.

## Features

- **Neural Network Visualization**: Interactive canvas-based neural network background with cursor-reactive nodes and connections
- **Professional Design**: Clean white theme with blue accents, responsive across all devices
- **Social Links**: Direct links to GitHub, LinkedIn, and email
- **Blog Integration**: Ready for daily newsletter automation
- **Static Export**: Optimized for GitHub Pages deployment with automatic GitHub Actions builds

## Tech Stack

- **Framework**: Next.js 14 with React 18
- **Language**: TypeScript
- **Styling**: CSS with Inter font
- **Visualization**: HTML5 Canvas for neural network
- **Deployment**: GitHub Pages via GitHub Actions

## Getting Started

### Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Production Build

```bash
npm run build
```

Generates static files in the `out/` directory ready for GitHub Pages.

## Deployment

The site automatically deploys to [https://jbrown54-oss.github.io](https://jbrown54-oss.github.io) when changes are pushed to the `main` branch via GitHub Actions.

## Blog Integration

Blog posts can be:
1. Added as markdown files in `app/blog/`
2. Dynamically generated by the daily newsletter automation at 7pm
3. Viewed at `/blog/[slug]` where slug matches the post identifier

To add a new post, add it to the `blogPosts` object in `app/blog/[slug]/page.tsx` and export it in `generateStaticParams()`.

## Customization

- **Colors**: Edit accent color in `styles/globals.css` (currently `#4a90e2`)
- **Neural Network**: Adjust node count, animation speed, and connection distance in `lib/neural-network.ts`
- **Social Links**: Update URLs in `components/SocialLinks.tsx`
- **Bio & Content**: Edit text in `app/page.tsx`
